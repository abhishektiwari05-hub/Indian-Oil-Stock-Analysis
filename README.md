# Indian Oil Stock Price Analysis

## Project Overview
This project analyzes the historical stock price data of Indian Oil Corporation (IOC) using SQL, Python, and Power BI. It includes data cleaning, exploratory data analysis (EDA), SQL analysis, and an interactive Power BI dashboard to identify stock trends and generate business insights.

## Tools Used
- SQL (MySQL)
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Power BI
- Jupyter Notebook

## Key Features
- Data Cleaning
- SQL Data Analysis
- Exploratory Data Analysis (EDA)
- Stock Price Trend Analysis
- Daily Returns Analysis
- Moving Average Analysis
- Interactive Power BI Dashboard

## Project Files
- Dataset (.csv)
- SQL Queries (.sql)
- Python Notebook (.ipynb)
- Power BI Dashboard (.pbix)

## Key Insights
- Analyzed historical stock price trends.
- Identified changes in trading volume.
- Calculated daily returns and moving averages.
- Built an interactive dashboard for data visualization.
